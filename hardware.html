E-Bike Learning

One of my projects during my time at Talkout was to create a learning module for a client. The first system I created was a universal quiz system where you could see all the questions and answers in the inspector and change them as needed. I did this by using a variation of classes that made up the quiz. It would then use a controller to read the serialized data from the inspector and show it in-game view. Any other developer who needed to change this had access too: question, answer, time, audio, and because it was implemented using SOLID principles this is expandable by any other developer. Additionally I created module 2, which was for diagnostic. They would be guided through a series of actions to ensure the bike was in good condition, showing clear instructions.
